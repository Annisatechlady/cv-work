1. What is the difference between var, let and const in javascript? Provide samples

  1. Var is a function-scoped, meaning it is accessible throughout the entire function in which it is declared
  2. Variables declared with var can be reassigned.
  3. Var is hoisted to the top os its scope.
  4. Let is blocked-scoped, meaning it is only accessible within the block, enclsoded by{}) where it is declared
  5. Variables declared with Let can be raessigned.
  6. Let is also Hoisted, but it is not initialized
  7. Const is blocke-scoped, similar to let.
  8. Variables declared with const cannot be reassigned.
  9. Const behaves like Let in terms of hoisting.
